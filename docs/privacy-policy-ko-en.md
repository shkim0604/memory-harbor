## 3. Permissions We Use / 사용 권한 안내

**EN**
- `CAMERA`: Used when users choose to capture or upload photos/content.
- `RECORD_AUDIO`: Used for in-app voice calling features.
- `POST_NOTIFICATIONS`: Used to deliver call and service notifications.
- Storage-related permissions: Used to save or upload user-selected files.

**KO**
- `CAMERA`: 사용자가 사진/콘텐츠 촬영 또는 업로드를 선택할 때 사용됩니다.
- `RECORD_AUDIO`: 앱 내 음성 통화 기능 제공을 위해 사용됩니다.
- `POST_NOTIFICATIONS`: 통화 및 서비스 알림 제공에 사용됩니다.
- 저장소 관련 권한: 사용자가 선택한 파일 저장/업로드에 사용됩니다.

We only use permissions for the purposes described above.
